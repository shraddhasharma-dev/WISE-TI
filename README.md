# Wise@TI Hackathon Platform Setup

## Tech Stack

* Frontend: React + Vite + TailwindCSS
* Backend: Node.js + Express
* Database: PostgreSQL
* ORM: Prisma
* Queue System: Redis + BullMQ
* AI: OpenAI API
* Email Service: SendGrid
* Docker for local infrastructure
